# 📘 Assignment: Data Structures in Python

## 🎯 Objective

Practice working with Python data structures by using lists, dictionaries, sets, and tuples to organize, update, and inspect data.

## 📝 Tasks

### 🛠️ Lists and Dictionaries

#### Description
Create data collections for a small student record system and write code that adds, updates, and reads values from them.

#### Requirements
Completed program should:

- Create a list of at least three student names.
- Create a dictionary that stores information for at least one student.
- Add a new item to the list.
- Update a value in the dictionary.
- Print the final list and dictionary contents.

### 🛠️ Sets and Tuples

#### Description
Use sets and tuples to manage unique values and fixed collections of information.

#### Requirements
Completed program should:

- Create a set of unique course topics.
- Add at least one new topic to the set.
- Create a tuple that stores a student's name and grade.
- Access and print values from the tuple.
- Show that the set does not store duplicate values.
